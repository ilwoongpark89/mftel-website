# MFTEL Dashboard — 마일스톤 로드맵

> 비전: 연구실/학술팀에 최적화된 세계 최고의 팀 대시보드 → SaaS 서비스
> 차별화: 노션("만능이지만 복잡") / 리니어("개발팀 전용")과 달리 연구 워크플로우에 최적화된 깔끔한 경험
> 기술 스택: Next.js + Supabase + Upstash Redis
> 최종 갱신: 2026-02-14

---
